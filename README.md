# deep-disk-cleanup
Deep Disk Cleanup —  bat-скрипт для автоматической очистки Windows: временные файлы, корзина, кэш обновлений, логи, DNS и браузер. Требует прав администратора.


# Возможности
Очистка временных файлов Windows и пользователя (%WINDIR%\Temp, %TEMP%)
Полная очистка корзины на системном диске ($Recycle.bin)
Удаление кэша загруженных обновлений с перезапуском службы Центра обновлений
Очистка всех журналов событий, кроме Security (сохранение аудита)
Сброс кэша DNS через ipconfig /flushdns
Очистка кэша браузера Microsoft Edge (Chromium)

# Установка
```bash
git clone https://github.com/<твой-username>/deep-disk-cleanup.git
cd deep-disk-cleanup
```
Никаких дополнительных зависимостей не требуется — скрипт работает на любой версии Windows.

# Запуск
```bash
# Обязательно от имени администратора!
deep_disk.cmd
```
Либо нажмите ПКМ на файле → Запуск от имени администратора. Скрипт последовательно выполнит все 6 этапов очистки, каждый этап сопровождается статусом выполнения.
